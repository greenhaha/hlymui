## API

| 属性     | 说明             | 类型                   | 默认值    |
| -------- | ---------------- | ---------------------- | --------- |
| type     | 设置按钮类型     | `primary` \| `default` | `default` |
| onClick  | 点击按钮时的回调 | (event) => void        | -         |
| disabled | 设置按钮禁用     | boolean                | false     |
